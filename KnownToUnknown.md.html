It’s so easy to get stuck when preparing for coding interview. It’s so easy to get overwhelmed.
And one of the most common problems I see is students trying to jump straight to Level 50 without putting in the work to get there.
If you’re struggling on Easy Leetcode problems, of course you’re going to struggle on the hard ones.
If you don’t understand DFS, how are you going to understand minimum spanning trees or max flow algorithms?
If you go too far out of your comfort zone, you’re going to struggle NO MATTER WHAT.
So today I want to give you a strategy that you can apply to whatever you’re learning, whether it’s a new data structure, a specific algorithm, or how to solve a random interview question:

Start with the known and build into the unknown.
Here’s how this works.
Let’s imagine that you want to learn about DFS and you know nothing about it.
First, you start with the simplest version. Simply take a tree and determine whether or not it contains a specific node.
Essentially, implement boolean containsValue(TreeNode root, int val).
You can start by looking at existing solutions. Maybe you walk through the code line by line. You do whatever it takes to understand the code as best as you can.
And what do you do then?
You implement a SMALL variation on this.
Maybe in addition to finding whether the value is in the tree you also want to print out all of the nodes you’ve visited. Maybe you want to find the path.
And then you keep iterating.
Try printing out the values in different orders (post-order, pre-order, post-order, etc).
Try finding the path between two different nodes in the tree.
What if rather than a tree it's a graph?
As you work through each variation, it forces you to better understand exactly how DFS actually works.
No more memorizing the code. Now you actually can create it from scratch.

Now this technique doesn’t just work for DFS. This simple strategy works for anything you are trying to learn.
The key is to start with what you DO know and slowly work your way into the unknown.

I encourage you to test this out for yourself and let me know how it works for you!
